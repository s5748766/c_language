Visual Studio 단축
1. 일반 단축키
프로젝트 및 파일을 다루거나 기본적인 작업을 할 때 자주 사용되는 단축키입니다.
 Ctrl + S: 현재 문서 저장.
 Ctrl + Shift + S: 모든 문서 저장.
 Ctrl + Z: 마지막 작업 실행 취소.
 Ctrl + Y: 실행 취소된 작업 다시 실행.
 Ctrl + Shift + N: 새 프로젝트 열기.
 Ctrl + O: 파일 열기.
 Ctrl + F4: 현재 문서 닫기.
 Ctrl + Shift + F4: 모든 문서 닫기.
 Ctrl + P: 현재 파일 인쇄.

2. 편집 단축키
코드를 작성하고 수정할 때 유용한 단축키들로, 작업 속도를 크게 향상시킬 수 있습니다.
 Ctrl + X: 선택한 텍스트 잘라내기.
 Ctrl + C: 선택한 텍스트 복사.
 Ctrl + V: 복사 또는 잘라낸 텍스트 붙여넣기.
 Ctrl + A: 문서 전체 선택.
 Ctrl + D: 현재 줄 또는 선택 영역 복제.
 Ctrl + K, Ctrl + C: 선택한 코드 블록 주석 처리.
 Ctrl + K, Ctrl + U: 선택한 코드 블록 주석 해제.
 Ctrl + Tab: 열린 문서 간 전환.
 Ctrl + Shift + L: 현재 줄 삭제.

3. 탐색 단축키
코드베이스를 탐색할 때 빠르게 이동할 수 있는 단축키입니다. 특히 복잡한 프로젝트에서 시간을 절약
해줍니다.
 Ctrl + T: 형식, 파일, 멤버에 대한 빠른 검색.
 F12: 선택한 심볼의 정의로 이동.
 Ctrl + F12: 선택한 심볼의 선언으로 이동.
 Ctrl + : 이전 위치로 이동.
 Ctrl + Shift + : 다음 위치로 이동.
 Ctrl + M, Ctrl + O: 문서 내 모든 코드 영역 축소.
 Ctrl + M, Ctrl + P: 문서 내 모든 코드 영역 확장.
 Ctrl + G: 특정 줄로 이동.

4. 디버깅 단축키
디버깅 과정에서 코드를 한 줄씩 확인하거나, 변수 상태를 추적할 때 유용한 단축키들입니다.
 F5: 디버깅 시작.
 Shift + F5: 디버깅 중지.
 F9: 현재 줄에 중단점 설정 또는 해제.
 F10: 현재 줄의 코드 건너뛰기.
 F11: 현재 줄의 함수/메서드 내부로 진입.
 Shift + F11: 현재 함수에서 나가기.
 Ctrl + Alt + I: 즉시 실행 창 열기.
 Ctrl + Alt + U: 호출 스택 창 열기.

5. 프로젝트 및 솔루션 관리 단축키
프로젝트 파일과 솔루션을 효율적으로 관리하는 데 유용한 단축키들입니다.
 Ctrl + Shift + A: 프로젝트에 새 항목 추가.
 Ctrl + Shift + O: 프로젝트 열기.
Ctrl + Shift + B: 솔루션 빌드.
 Alt + Enter: 선택한 항목의 속성 창 열기.
 F7: 디자인 뷰와 코드 뷰 간 전환.

6. 창 관리 단축키
복잡한 프로젝트를 다룰 때 여러 창을 관리하는 일이 중요합니다. 아래 단축키들은 비주얼 스튜디오의
창을 더 쉽게 관리할 수 있도록 도와줍니다.
 Ctrl + Shift + F4: 모든 열린 창 닫기.
 Ctrl + Shift + W: 현재 창 닫기.
 Shift + Alt + Enter: 전체 화면 모드 전환.
 Ctrl + Alt + L: 솔루션 탐색기 창 열기.
 Ctrl + Alt + X: 도구 상자 창 열기.
 Ctrl + ,: 열린 파일 간 이동.

7. 검색 및 바꾸기 단축키
대규모 코드베이스에서 특정 단어나 구문을 빠르게 찾거나 변경할 때 유용한 단축키입니다.
 Ctrl + F: 찾기 창 열기.
 Ctrl + H: 찾기 및 바꾸기 창 열기.
 Ctrl + Shift + F: 파일 내에서 찾기 창 열기.
 Ctrl + Shift + H: 파일 내에서 바꾸기 창 열기.
 F3: 다음 검색 결과로 이동.
 Shift + F3: 이전 검색 결과로 이동.

8. 코드 리팩터링 단축키
코드의 가독성과 유지보수성을 높이기 위해 리팩터링할 때 유용한 단축키들입니다.
 Ctrl + R, R: 선택한 변수, 클래스 또는 메서드 이름 변경.
 Ctrl + K, Ctrl + D: 문서 포맷 정리.
 Ctrl + .: 빠른 작업 및 리팩터링 옵션 표시.
 Ctrl + K, Ctrl + X: 코드 스니펫 삽입.
 Ctrl + K, Ctrl + E: 선택한 코드를 스니펫으로 감싸기.
